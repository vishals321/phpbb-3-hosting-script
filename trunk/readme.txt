** Installation **
- Das Script in den Dokumenten-Root kopieren
- Eine Wilcard-Subdomain per Confixx, Plesk oder anderem auf das Verzeichnis phpBB3 legen
- Das Script aufrufen
- Installationsroutine befolgen
- Order install löschen
- config.php CHMOD auf 644 setzen
- in den Admin bereich unter /admin/ aufrufen und das Script konfigurieren
- Fertig

Das Script bietet folgende Funktionen:

    * Template-System vom phpBB
    * Language-System
    * Statistik
    * Administrations-Bereich im Hosting Script
    * User-Login
    * Kategorien-System
    * FAQ-System
    * config.php der Foren wird beim ersten Aufruf eines Forums automatisch ausgelesen und in eine Datei für jedes Forum einzelnd geschrieben.
    * Logo-Upload Funktion im Forum
    * Kontaktformular
    * Konfigurierbares Impressum
    * Multidatenbankfähig (Bei der Erstellung eines Forums wird immer die Datenbank ausgewählt in der derzeit die wenigsten Foren sind)
    * Deutsche AGB bereits gegeben
    * Administrationsbereich vom normalen Bereich getrennt
    * Administrationsbereich kann per .htaccess geschützt werden
    * Administrationsbereich mit mehreren Administratoren möglich

Benötigt werden

    * Wildcard-Subdomains
    * PHP 5 fähiger Server
    * MySQL Datenbank (am besten mehrere)

Derzeit ist das Script nur in deutsch verfügbar.
